# Projeto Exemplo - Site Institucional para Vercel

Projeto simples em HTML + CSS para primeiro deploy na Vercel.

## Objetivo didático

- praticar versionamento básico;
- publicar em URL pública;
- entender deploy automático por GitHub.

## Como usar em aula

1. Abra esta pasta no VS Code.
2. Edite textos para personalizar o tema do site.
3. Faça commit e publique no GitHub.
4. Importe o repositório na Vercel e faça deploy.

## Estrutura

- `index.html`
- `styles.css`

## Observação

Como o projeto é estático, a Vercel detecta automaticamente e publica sem configuração extra.
