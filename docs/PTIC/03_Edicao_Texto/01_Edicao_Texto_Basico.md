# Edição de Texto: O Básico

> Aula prática para aprender digitação, formatação de fonte, parágrafos e cores em editor de texto.

---

## Introdução

Nesta aula, você vai aprender o mínimo essencial para criar um texto organizado e fácil de ler.

O foco é dominar 4 pontos básicos:

- Digitação com atenção.
- Fonte (tipo, tamanho e estilo).
- Parágrafo (alinhamento e espaçamento).
- Cor de texto e destaque.

---

## O que vamos aprender

- Abrir um editor de texto (Google Docs ou Word).
- Digitar um pequeno texto sem perder a organização.
- Aplicar negrito, itálico e sublinhado corretamente.
- Ajustar tamanho da fonte e alinhamento dos parágrafos.
- Usar cor com bom senso para destacar informação importante.

---

## Regras simples de digitação

Antes de formatar, o texto precisa estar bem escrito:

- Comece frases com letra maiúscula.
- Termine frases com ponto final.
- Use um espaço entre palavras.
- Evite escrever tudo em MAIUSCULO.
- Revise erros de digitação antes de enviar.

!!! tip "Dica"
    Escrever bem primeiro e formatar depois evita retrabalho


---

## Formatação de fonte

A fonte muda a aparência do texto.

### Principais recursos

- **Tipo de fonte:** exemplo Arial, Calibri, Times New Roman.
- **Tamanho:** normalmente 11 ou 12 para texto comum.
- **Negrito:** destacar título ou palavra-chave.
- **Itálico:** destacar termo estrangeiro ou observação.
- **Sublinhado:** usar pouco, apenas quando necessário.

### Atalhos úteis

- `Ctrl + B`: negrito.
- `Ctrl + I`: itálico.
- `Ctrl + U`: sublinhado.

---

## Formatação de parágrafo

Parágrafo bem ajustado deixa a leitura mais clara.

### Ajustes básicos

- **Alinhamento:**
  - Esquerda (mais comum para textos escolares).
   - Centralizado (normalmente para títulos).
  - Justificado (visual mais alinhado nas laterais).
- **Espaçamento entre linhas:** use 1,15 ou 1,5 para facilitar a leitura.
- **Espaço entre parágrafos:** separa ideias e evita bloco de texto muito grande.

---

## Cores no texto

Cor serve para orientar o leitor, não para enfeitar demais.

Use assim:

- Preto para texto principal.
- Uma cor de destaque para palavra importante.
- Evite muitas cores no mesmo parágrafo.
- Garanta contraste (texto escuro em fundo claro).

---

## Prática guiada

1. Abra o Google Docs ou Word.
2. Crie um arquivo com nome `PTIC_Aula06_Edicao_Basica`.
3. Digite o título: `Minha primeira formatacao de texto`.
4. Digite dois parágrafos curtos sobre sua rotina de estudos.
5. Formate o título com:
   - Negrito.
   - Tamanho 16.
   - Alinhamento centralizado.
6. No corpo do texto, aplique:
   - Fonte tamanho 12.
   - Alinhamento à esquerda.
   - Espaçamento 1,15.
7. Destaque 3 palavras importantes com negrito.
8. Troque a cor de 1 frase para azul escuro.
9. Revise e salve.
