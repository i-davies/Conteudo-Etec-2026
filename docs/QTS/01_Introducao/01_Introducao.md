# Qualidade e Testes de Software (QTS)

> Material didático para a disciplina de Qualidade e Testes de Software.

---

## Introdução

Bem-vindo à disciplina de **QTS**. Garantir que o software funcione conforme o esperado é vital, e aqui você aprenderá como medir e assegurar essa qualidade.

## Objetivos

- Conhecer os diferentes tipos de testes (Unitários, Integração, E2E).
- Aplicar técnicas de testes automatizados.
- Entender métricas de qualidade de software.

---

!!! tip "Dica"
    Um software sem testes é um software pronto para falhar no pior momento possível.
