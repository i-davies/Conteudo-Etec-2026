# Banco de Dados 2 (BD2)

> Material didático para a disciplina de Banco de Dados 2.

---

## Introdução

Bem-vindo à disciplina de **Banco de Dados 2**. Aqui exploraremos conceitos avançados de modelagem, automação e otimização de sistemas de banco de dados utilizando **MySQL**.

## O que vamos aprender

Nossa carga horária de 80h está dividida em quatro grandes etapas:

1. **Estrutura e Definição (DDL):** Criação de bancos, tabelas e regras de integridade.
2. **Manipulação e Consultas (DML e DQL):** Inserção, edição e buscas avançadas com filtros e junções (Joins).
3. **Automação e Lógica:** Criação de Visões (Views), Procedures, Funções e Gatilhos (Triggers).
4. **Otimização e Segurança:** Uso de Índices, análise de performance e controle de usuários.
