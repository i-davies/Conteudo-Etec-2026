# Atividade Avaliativa em Sala

> Esta atividade vale nota e deve ser feita individualmente em sala de aula.

---

## Tema da Atividade

Desenvolver um site com o tema **Guia de Pontos Turísticos da Cidade**.

## Objetivo

Aplicar os conteúdos estudados em PW1 nas semanas `07` a `12` (semântica, CSS com seletores e cores, tipografia, box model, backgrounds e posicionamento com `float` e `display`), construindo um site visualmente organizado, com estrutura clara e estilização coerente.

## Requisitos obrigatórios

- Criar um site com **2 páginas HTML**:
    - `index.html`: página principal do guia turístico;
    - `roteiro.html`: página com roteiro e destaques dos pontos turísticos.
- O projeto deve usar corretamente elementos semânticos, incluindo:
    - `header`;
    - `main`;
    - `section`;
    - `footer`.
- A página inicial deve conter obrigatoriamente:
    - um banner principal com `background` ou gradiente;
    - um `h1`;
    - pelo menos 2 `h2`;
    - parágrafos com informações sobre os pontos turísticos escolhidos.
- A página `roteiro.html` deve conter obrigatoriamente:
    - uma lista ou tabela com locais, horários e informações úteis de visitação;
    - pelo menos 3 blocos de conteúdo organizados com `display` (`block`, `inline`, `inline-block` ou combinação adequada);
    - pelo menos 1 exemplo de bloco lateral usando `float` (`left` ou `right`), com conteúdo textual;
    - links funcionais para voltar à página inicial e ao topo.
- O CSS deve contemplar obrigatoriamente:
    - seletores por tag, classe ou `id`;
    - cores de texto e de fundo;
    - uso de fontes do Google Fonts;
    - aplicação de `margin`, `padding` e `border`;
    - uso de `background-color` ou gradiente.

??? tip "Organização sugerida"
    Monte primeiro a estrutura semântica das duas páginas. Depois aplique cores, tipografia e espaçamentos. Deixe o uso de `float` e `display` para a etapa final de organização visual.

## Cenário

Você foi contratado para criar uma versão simples de um **guia turístico local**.

O objetivo é apresentar os principais pontos turísticos da cidade de forma visualmente agradável, com informações claras para visitantes.

## Critérios de Correção

- 2,0 pts: estrutura correta das 2 páginas com uso adequado de HTML semântico.
- 2,0 pts: aplicação correta de CSS com seletores, cores e tipografia.
- 2,0 pts: uso adequado de `margin`, `padding`, `border` e organização visual dos blocos.
- 2,0 pts: uso correto de `background-color` ou gradiente e composição visual coerente com o tema.
- 2,0 pts: navegação funcionando, uso correto de `float` e `display`, além de organização e legibilidade do código.

## Entrega

- Compactar a pasta do projeto em `.zip`.
- Enviar via Teams com o nome: `AT2_PW1_1C_T1_nome_sobrenome.zip`.
- O arquivo deve conter as páginas HTML e os demais arquivos necessários para o site abrir corretamente.

## Observações

- O site deve funcionar corretamente no navegador ao abrir o arquivo `index.html`.
- Projetos sem as 2 páginas ou sem CSS aplicado não atendem aos requisitos mínimos da atividade.
