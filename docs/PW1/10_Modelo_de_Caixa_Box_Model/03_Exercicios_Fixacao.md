# Exercício Integrador: Tipografia e Box Model

Neste exercício, você vai juntar os dois conteúdos mais recentes:

- uso de fontes com Google Fonts;
- organização visual com `margin`, `padding` e `border`.

## Proposta

Crie ou atualize uma página com o tema **Festival Cultural da Escola**.

A página deve apresentar informações sobre um evento fictício da escola, com destaque para atrações, horários e convite ao público.

## Requisitos do Exercício

Sua página deve conter:

- `header` com título e subtítulo;
- `main` com pelo menos três seções;
- uma imagem com `alt` e legenda;
- uma tabela com programação ou horários;
- duas fontes importadas do Google Fonts;
- uma fonte aplicada aos títulos e outra ao corpo do texto;
- uso visível de `padding` em blocos de conteúdo;
- uso visível de `border` em pelo menos dois elementos;
- uso visível de `margin` para separar seções.

## Orientações de Estilo

- Use cores vivas nas bordas para deixar o Box Model perceptível.
- Mantenha boa leitura nos parágrafos.
- Evite exagerar na quantidade de fontes e cores.
- Faça a página parecer organizada e convidativa.

??? example "Estrutura sugerida"
    Você pode organizar a página desta forma:

    - `header` com nome do festival e frase de abertura.
    - Uma seção com apresentação do evento.
    - Uma seção com imagem e legenda.
    - Uma seção com tabela de horários.
    - Uma seção final com convite ou destaque especial.

## Arquivo sugerido

- `PW1/10_aula/festival_cultural.html`

## Checklist

- [ ] A página usa duas fontes importadas do Google Fonts.
- [ ] Os títulos têm identidade visual diferente do corpo do texto.
- [ ] As seções têm `padding` para melhorar a leitura.
- [ ] Há `border` visível em elementos importantes.
- [ ] O uso de `margin` separa bem os blocos.
- [ ] A estrutura HTML continua semântica e organizada.
