# Exercícios de Fixação: Preparando a Tipografia da Página

Nesta etapa, você vai deixar sua página pronta para o exercício integrador da próxima aula.
O foco agora é organizar o uso de fontes e melhorar a leitura do conteúdo.

## Etapa 1

No seu projeto atual, importe duas fontes do Google Fonts.

- uma para títulos;
- uma para parágrafos e textos gerais.

## Etapa 2

Aplique ajustes tipográficos nos elementos abaixo:

- `h1` com fonte de destaque;
- `h2` com tamanho menor que o título principal;
- `p` com fonte legível e bom espaçamento entre linhas.

## Etapa 3

Revise o resultado visual da página:

- veja se os títulos se destacam;
- confira se os parágrafos estão fáceis de ler;
- mantenha coerência com as cores já usadas na aula anterior.

Na próxima pasta, você vai manter essa tipografia e acrescentar espaçamentos e bordas usando o Box Model.
