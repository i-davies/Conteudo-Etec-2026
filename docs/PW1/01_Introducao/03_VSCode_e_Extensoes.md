# Ferramentas: VS Code e Extensões

Para construir sites, não usamos editores de texto comuns (como o Bloco de Notas ou o Word). Nós usamos programas especiais chamados **Editores de Código**, que nos ajudam a escrever melhor e mais rápido, colorindo as palavras e apontando erros.

Neste curso, vamos usar o **Visual Studio Code (VS Code)**.

## O que é o VS Code?

O VS Code é como uma caixa de ferramentas para nós, desenvolvedores. Ele é gratuito, leve e muito poderoso.

- **Diferente de um editor normal:** Ele entende o que estamos escrevendo e sugere autocompletar as palavras.
- **Extensões:** Você pode instalar superpoderes nele, como corretores ortográficos ou servidores instantâneos.

## Preparando o Ambiente (Na Prática)

Siga com o professor os seguintes passos para deixar seu VS Code pronto:

- Abrir o **VS Code** no seu computador.
- Reconhecer a tela inicial e a barra lateral (onde ficam nossos arquivos).
- Instalar extensões mágicas!

### Extensões Recomendadas

No menu lateral do VS Code, procure pelo ícone de **Extensões** (ou aperte `Ctrl + Shift + X`). Busque e instale:

- **Live Server:** Cria um computador servidor temporário e recarrega a página automaticamente quando você salva o arquivo. (Sem precisar ficar apertando F5!).
- **Material Icon Theme:** Deixa os ícones das suas pastas e arquivos muito mais bonitos e fáceis de identificar.

## Exercícios de Fixação

Chegou a hora de criar nossa primeira pasta de trabalho e nosso primeiro arquivo!

??? example "Desafio Nosso Primeiro Código"
    Nesta atividade, você vai organizar seu espaço de trabalho e escrever sua primeira mensagem no navegador.
    
    **Passo a passo da atividade:**
    
    - Crie uma pasta para a aula de hoje seguindo o padrão da nossa disciplina: `PW1/02_aula/primeiro_codigo`.
    - Abra o **VS Code**, vá no menu lateral esquerdo superior e escolha a opção para abrir a pasta que você acabou de criar.
    - Dentro do VS Code, clique no ícone para **Novo Arquivo**.
    - Dê o nome de `index.html` para esse arquivo (a terminação `.html` é muito importante!).
    - Dentro do arquivo `index.html`, escreva a mensagem: `Olá Mundo, estou aprendendo a programar para a Web!`
    - Salve o arquivo (`Ctrl + S`).
    - Clique com o botão direito no arquivo e escolha **Open com Live Server**.
    
    Parabéns! O navegador deve abrir sozinho e mostrar a sua primeira página. Mostre ao colega do lado o seu resultado!
