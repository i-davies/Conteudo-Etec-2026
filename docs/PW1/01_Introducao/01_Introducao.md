# Programação Web I (PW1)

> Material didático para a disciplina de Programação Web I.

## O que é Programação para Web?

A Programação Web é a arte de construir tudo o que você vê e interage na internet. Nosso foco aqui será o desenvolvimento **Front-end**, ou seja, a parte visual, aquilo que aparece na tela do seu navegador.

Vamos aprender como construir páginas estruturadas, bonitas e que interagem com o usuário.

## Nossa Jornada no Curso

Durante o ano, vamos passar pelas seguintes etapas para nos tornarmos desenvolvedores web:

- **1º Bimestre: O Esqueleto (HTML5)**
  Foco: Entender como criamos a estrutura do site. É a planta baixa da nossa página.
- **2º Bimestre: A Aparência (CSS3 - Básico)**
  Foco: Dar cor, tamanho e posição para as coisas. Vamos deixar tudo bonito!
- **3º Bimestre: Layout e Interatividade (CSS Avançado e JS Inicial)**
  Foco: Criar layouts profissionais e dar os primeiros passos para a página ganhar vida.
- **4º Bimestre: Dinamismo e Projeto Final (JavaScript)**
  Foco: Fazer a página pensar e reagir ao usuário. Colocar tudo em prática num grande projeto!

!!! tip "O Segredo do Sucesso"
    Para aprender programação, a prática é fundamental. Você não vai quebrar o computador por errar o código, então experimente, modifique e veja o que acontece na tela!
