# Programação Web I (PW1)

> Material didático para a disciplina de Programação Web I.

## O que é Programação para Web?

A Programação Web é a arte de construir tudo o que você vê e interage na internet. Quando você acessa suas redes sociais, assiste a vídeos ou joga online no navegador, está usando o resultado do trabalho de desenvolvedores web.

Nosso foco aqui será o desenvolvimento **Front-end**, ou seja, a parte visual e interativa dos sites, aquilo que roda diretamente no navegador do usuário.

## O que você já sabe sobre a Web?

Antes de começarmos a codificar, vamos fazer um pequeno teste para ver o que você já conhece sobre o mundo do desenvolvimento web. Pense nas respostas e depois expanda os blocos para conferir!

??? question "O que significa a sigla WWW?"
    **World Wide Web** (Teia Mundial de Computadores). É o sistema interligado de documentos e recursos que acessamos pela internet usando navegadores.

??? question "Qual a diferença entre a internet e a web?"
    A **Internet** é a infraestrutura física (cabos, roteadores e computadores conectados). A **Web** é um dos serviços que funcionam por cima da internet, focada no acesso a páginas e sites.

??? question "Você sabe quais são as 3 principais linguagens do Front-end?"
    São elas: **HTML** (estrutura), **CSS** (estilo/aparência) e **JavaScript** (comportamento/interatividade).

## Nossos Objetivos

Durante o curso, nosso objetivo é sair do zero e construir sites profissionais, passando pelos seguintes pilares:

- **O Esqueleto (HTML5):** Aprender a estruturar os componentes e o conteúdo de uma página.
- **A Aparência (CSS3):** Dominar a estilização, entender o modelo de caixa e criar layouts bonitos e responsivos.
- **Layout e Interatividade:** Dar os primeiros passos com CSS Avançado e JavaScript Inicial para trazer responsividade e dinamismo.
- **Dinamismo e Projeto Final:** Manipular a página com JavaScript e consolidar o aprendizado construindo um projeto completo do início ao fim.

!!! tip "O Segredo do Sucesso"
    Para aprender programação, a prática é fundamental. Você não vai quebrar o computador por errar o código, então experimente, modifique e veja o que acontece na tela!
