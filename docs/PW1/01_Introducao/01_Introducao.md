# Programação Web 1 (PW1)

> Material didático para a disciplina de Programação Web 1.

---

## Introdução

Bem-vindo à disciplina de **Programação Web 1**. Aqui daremos os primeiros passos no desenvolvimento de sites, focando nas tecnologias base da Web.

## Objetivos

- Aprender HTML5 semântico.
- Dominar estilização com CSS3.
- Introdução à lógica de programação com JavaScript.

---

!!! tip "Dica"
    O domínio do HTML e CSS é a base para qualquer desenvolvedor web de sucesso.
