# Exercício Integrador: Backgrounds e Gradientes

Neste exercício, você vai aprimorar o projeto do Festival Cultural com banners visuais usando gradientes.

## Proposta

Atualize a página do Festival Cultural (que já tem estrutura, fontes e box model) adicionando backgrounds e gradientes em:

- um banner principal com destaque;
- seções com backgrounds diferenciados;
- possíveis efeitos visuais complementares.

## Requisitos do Exercício

Sua página atualizada deve conter:

- um banner principal com gradiente linear (mínimo 2 cores);
- pelo menos dois gradientes diferentes aplicados em seções ou elementos;
- um uso de gradiente radial OU uma sobreposição de gradiente com imagem;
- texto com bom contraste contra o background (legível);
- padding adequado em elementos com background para dar respiro ao conteúdo;
- consistência visual entre os gradientes escolhidos;
- manutenção de toda a estrutura anterior (estrutura HTML semântica, fontes, box model).

## Orientações de Estilo

- Escolha cores que reflitam o tema do festival (vibrante, cultural, atrativo).
- Use no máximo 3 cores por gradiente para não ficar carregado.
- Teste a legibilidade do texto sobre cada background.
- Mantenha os gradientes alinhados com a identidade visual já criada.
- Evite gradientes muito chamativos que distraiam do conteúdo.

??? example "Estrutura sugerida"
    Você pode organizar assim:

    - `header` com banner principal (gradiente linear diagonal).
    - Seção de apresentação do festival com background suave.
    - Seção com imagem e gradiente sobreposto.
    - Seção com programação ou atrações usando backgrounds alterados.
    - `footer` com um gradiente de encerramento.

## Checklist

- [ ] O banner principal tem um gradiente linear visível.
- [ ] Há pelo menos dois gradientes diferentes na página.
- [ ] O texto é legível sobre todos os backgrounds.
- [ ] Existe padding suficiente em elementos com background.
- [ ] Os gradientes criam harmonia visual e não distraem do conteúdo.
- [ ] A página mantém toda a estrutura semântica anterior.
- [ ] As fontes e cores originais foram preservadas ou melhoradas.
