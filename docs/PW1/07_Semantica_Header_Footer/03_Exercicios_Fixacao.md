# Exercícios de Fixação: Semântica com `header` e `footer`

Pratique a criação de páginas HTML mais organizadas, usando estrutura semântica com textos, imagens e tabelas.

## Exercício 1

Crie uma página com tema livre contendo:

- `header` com título e subtítulo;
- `main` com duas seções de conteúdo em texto;
- `footer` com identificação do aluno e turma.

## Exercício 2

Na mesma página, adicione:

- uma imagem com `alt` dentro de `figure`;
- uma legenda com `figcaption`;
- uma tabela com pelo menos três colunas e duas linhas de dados.

## Exercício 3

Ajuste o conteúdo para ficar bem dividido:

- uma seção para apresentação do tema;
- uma seção para imagem;
- uma seção para tabela.
