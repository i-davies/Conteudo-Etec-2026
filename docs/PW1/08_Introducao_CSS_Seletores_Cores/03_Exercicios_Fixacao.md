# Exercícios de Fixação: CSS com Seletores e Cores

Nesta prática, você vai evoluir a página semântica da aula anterior aplicando estilos com CSS inline e CSS por seletores.

## Exercício 1

No seu projeto anterior, aplique as mudanças abaixo:

- cor de fundo para o `body`;
- cor de texto padrão da página;
- cor de fundo e cor de texto para `header`;
- cor de fundo e cor de texto para `footer`.

## Exercício 2

Crie e aplique pelo menos:

- uma classe para destacar um card ou bloco de conteúdo;
- um id para estilizar um elemento único da página;
- um exemplo de CSS inline em um único elemento de destaque.

## Exercício 3

Garanta que a página mantenha:

- estrutura semântica com `header`, `main`, `section` e `footer`;
- uma imagem com `alt` e legenda;
- uma tabela com dados do tema escolhido.

??? example "Desafio Projeto Completo"
    Crie um projeto chamado **Guia de Turismo da Sua Cidade** com tudo o que já foi aprendido até agora.

    **Arquivo esperado:**

    - `PW1/08_aula/css_cores/projeto_turismo.html`

    **Requisitos mínimos:**

    - Estrutura semântica completa.
    - Pelo menos 2 seções com conteúdo textual.
    - 1 imagem com `alt` e `figcaption`.
    - 1 tabela com no mínimo 3 colunas e 3 linhas de dados.
    - Cores de fundo e texto aplicadas com seletores.
    - Pelo menos 1 uso de CSS inline para comparação.

## Checklist

- [ ] A página possui estrutura semântica organizada.
- [ ] Existem estilos aplicados com seletores.
- [ ] Há pelo menos um exemplo de CSS inline.
- [ ] As cores de fundo e texto melhoraram a leitura.
- [ ] O tema escolhido está completo com texto, imagem e tabela.
