# Exercício Integrador: Posicionamento com Float e Display

Neste exercício, você vai aperfeiçoar o layout do Festival Cultural usando técnicas de posicionamento para alinhar imagens e texto de forma profissional.

## Proposta

Atualize a página do Festival Cultural (que já tem estrutura, fontes, box model e backgrounds) implementando layouts com imagens ao lado de texto usando float e propriedades de display.

## Requisitos do Exercício

Sua página atualizada deve conter:

- pelo menos duas seções com imagem e texto lado a lado;
- uso correto de float em uma seção e de `display: inline-block` em outra seção;
- espaçamento adequado entre imagem e texto com margin;
- alinhamento vertical apropriado (topo, centro ou responsivo);
- consistência no padrão de layout ao longo da página;
- responsividade básica (em mobile, imagens podem ficar acima do texto);
- manutenção de toda a estrutura anterior (HTML semântico, fontes, box model, backgrounds).

## Orientações de Estilo

- Use imagens com tamanhos consistentes em todo o projeto.
- Adicione `border-radius` nas imagens para um toque moderno.
- Mantenha o contraste e legibilidade do texto.
- Teste o layout em diferentes tamanhos para garantir que funciona bem.
- Evite imagens muito grandes que prejudiquem a leitura em mobile.

??? example "Estrutura sugerida"
    Organize assim:

    - `header` com banner (como na aula anterior).
    - Seção 1: Imagem à esquerda, texto à direita (apresentação do festival).
    - Seção 2: Imagem à direita, texto à esquerda (detalhe de uma atração).
    - Seção 3: Tabela ou lista de atrações (sem imagem ao lado).
    - `footer` com informações adicionais.

    Dica: Alterne a posição das imagens (esquerda, depois direita) para criar ritmo visual.

## Arquivo sugerido

## Checklist

- [ ] Há pelo menos duas seções com imagem e texto lado a lado.
- [ ] As imagens têm espaçamento adequado com margin.
- [ ] O layout aplica float em uma seção e `display: inline-block` em outra.
- [ ] O texto é legível e bem posicionado em relação à imagem.
- [ ] O alinhamento vertical é apropriado (não parece desalinhado).
- [ ] As imagens têm `alt` descritivo e estão bem dimensionadas.
- [ ] A página mantém coerência visual com banners e backgrounds anteriores.
- [ ] O layout funciona razoavelmente bem em mobile (imagens acima do texto é aceitável).
