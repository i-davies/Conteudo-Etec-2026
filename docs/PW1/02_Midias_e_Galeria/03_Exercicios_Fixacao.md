# Exercicios de Fixacao: Galeria de Fotos

Agora voce vai construir uma galeria completa para consolidar o uso de imagens e figuras.

??? example "Desafio Galeria de Tema Livre"
	Crie uma galeria com um tema de interesse da turma.

	**Estrutura esperada:**

	- `PW1/04_aula/galeria/index.html`
	- `PW1/04_aula/galeria/imagens/...`

	**Requisitos minimos:**

	- Titulo principal da galeria com `<h1>`.
	- No minimo 6 imagens.
	- Todas as imagens com `alt`.
	- Pelo menos 3 blocos com `<figure>` e `<figcaption>`.
	- Um link interno para voltar ao topo da pagina.

	**Entrega:**

	- Arquivo `index.html` funcionando no navegador.
	- Codigo organizado e identado.

## Checklist

- [ ] Tema escolhido.
- [ ] Seis imagens inseridas.
- [ ] Textos alternativos preenchidos.
- [ ] Legendas aplicadas.
- [ ] Link interno funcionando.
