## Exercícios de escrita

### Dissertativa 1 - Estrutura semântica básica

Escreva um trecho HTML para uma página de "Clube de Leitura" contendo:

- um header com título e frase curta de apresentação;
- um main com duas seções;
- na primeira seção, um subtítulo e um parágrafo;
- na segunda seção, um subtítulo e outro parágrafo;
- um footer com nome do aluno e turma.

### Dissertativa 2 - Seletores por classe e id

Crie um pequeno exemplo com HTML e CSS em que:

- exista um título com id;
- existam dois parágrafos com a mesma classe;
- o id receba uma cor de texto;
- a classe receba cor de fundo e padding.

### Dissertativa 3 - Box Model

Escreva CSS para a classe `.bloco` aplicando:

- `padding` de 16px;
- `border` sólida de 2px;
- `margin-bottom` de 12px;
- uma cor de fundo clara.

Depois, escreva um HTML mínimo com dois elementos usando a classe `.bloco`.

### Dissertativa 4 - Banner com gradiente

Escreva HTML e CSS para um banner de topo com:

- classe `.banner`;
- gradiente linear em diagonal;
- texto em branco;
- padding para dar espaço interno.

### Dissertativa 5 - Float com texto

Considere o HTML abaixo e escreva apenas o CSS:

```html
<section class="materia">
  <img src="imagens/livro.jpg" alt="Capa de livro">
  <p>Texto sobre a atividade de leitura da turma.</p>
</section>
```

No CSS, faça:

- a imagem flutuar à esquerda;
- definir largura da imagem;
- criar espaço entre imagem e texto.

### Dissertativa 6 - Inline-block com tags

Considere o HTML abaixo e escreva apenas o CSS:

```html
<p>
  <span class="tag">Música</span>
  <span class="tag">Teatro</span>
  <span class="tag">Dança</span>
</p>
```

No CSS da classe `.tag`, aplique:

- `display: inline-block`;
- borda simples;
- padding;
- margem à direita.
