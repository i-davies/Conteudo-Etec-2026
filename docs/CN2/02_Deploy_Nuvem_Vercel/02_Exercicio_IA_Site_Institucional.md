# Site Institucional com IA + Deploy Automático na Vercel

> Objetivo: gerar um site institucional com apoio de IA, versionar no GitHub e publicar automaticamente na Vercel.

---

## Cenário

Você foi contratado para criar um site institucional de uma empresa local (escola, clínica, oficina, loja, escritório etc.).

O cliente quer:

- página inicial com identidade visual;
- seção de serviços;
- seção "sobre";
- formulário de contato;
- site publicado em URL pública.

---

## Ferramentas sugeridas para gerar o site com IA

Você pode escolher uma (ou mais):

- **Lovable**
- **Google Stitch**
- **v0 (Vercel)**
- **Bolt.new**
- **Windsurf**
- **ChatGPT** (gerando estrutura HTML/CSS/JS)

Use prompts claros, por exemplo:

```text
Crie um site institucional moderno para uma clínica veterinária.
Use layout responsivo, com menu, banner principal, seção de serviços,
seção sobre, depoimentos e formulário de contato.
Entregue em HTML, CSS e JavaScript puro.
```

---

## Etapas do exercício

### Parte 1 - Gerar e baixar o projeto

1. Gere o site em uma ferramenta de IA.
2. Revise o conteúdo e ajuste textos para português.
3. Baixe/exporte o projeto para seu computador.
4. Abra a pasta no VS Code.

### Parte 2 - Subir para GitHub usando VS Code

1. Abra **Source Control**.
2. Clique em **Initialize Repository** (se necessário).
3. Faça o primeiro commit com mensagem clara.
4. Clique em **Publish to GitHub**.
5. Confirme se o repositório ficou público (ou privado, se orientado pelo professor).

### Parte 3 - Deploy automático na Vercel

1. Entre em <https://vercel.com> com GitHub.
2. Clique em **Add New...** > **Project**.
3. Importe o repositório criado.
4. Clique em **Deploy**.
5. Teste a URL gerada pela Vercel.

---

## Requisitos mínimos de avaliação

- [ ] Site institucional com no mínimo 4 seções.
- [ ] Layout responsivo (desktop e celular).
- [ ] Repositório no GitHub com histórico inicial de commit.
- [ ] Deploy funcionando na Vercel.
- [ ] URL pública entregue ao professor.

### Critérios de qualidade (diferencial)

- Clareza visual e legibilidade.
- Conteúdo coerente com o tipo de negócio.
- Organização de pastas e nomes de arquivos.
- Boa adaptação para telas menores.
- Pequenos ajustes manuais após geração por IA.
